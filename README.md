## Installation

Install my-project with npm

```bash
  git clone https://github.com/HichamDe/gestion-formation.git
```
```bash
  npm install
```
Create a file in the public folder and name it 'output.css'
```bash
  npm run build-css
```
to run the json-server as a database
```bash
  npm run db
```
```bash
  npm start
```
